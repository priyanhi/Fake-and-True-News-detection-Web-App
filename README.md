# Fake-and-True-News-detection-Web-App

This project is a machine learning-based web application that detects whether a news article is real or fake using natural language processing (NLP) techniques. The goal of this project is to address the growing problem of misinformation by providing a simple tool that can quickly analyze and classify news content.

The model is built using a supervised learning approach. A dataset containing labeled fake and real news articles is first preprocessed to remove noise such as punctuation, special characters, URLs, and unnecessary whitespace. All text data is converted to lowercase to maintain consistency.
